# Illustration prompts — Acne Decoded

Generate each image with your preferred AI image-gen tool (Midjourney v6+, DALL-E 3, Imagen 3, Recraft, Ideogram, etc.). Save the result as `.png` (1024×1024 minimum, 1:1 aspect) into the matching lesson folder. Lesson markdown already references the expected paths — drop the image in and it appears.

**House style anchor (re-use for every prompt):**

- Background: cream off-white `#f5f1ec`
- Soft yellow blob behind subject for warmth
- Multi-colour palette only: pink `#e697b7`, mint green `#a8d8c0`, warm yellow `#f5e8a8`, deep teal `#2a4d3e`, cream-light `#faf6f0`
- Chunky charcoal hand-drawn outlines (`#212121`)
- Solid colour shape fills — no gradients, no drop shadows, no glow effects
- 4-pointed sparkle stars in pink / mint / yellow scattered around the subject
- A few small dots and short charcoal rays
- One or two hand-drawn curve doodles for warmth
- Square 1:1 aspect, generous breathing room, slightly asymmetric composition
- Style: friendly, editorial, slightly playful, medical-themed
- Reference: dribbble medical-illustration sets — Folio Illustration Agency, Dani Pasquini, similar

**Universal negative prompt:**
> photorealistic, 3D rendering, gradient, drop shadow, glow effect, realistic photography, dark mode, sketchy, watercolour, text, labels, logos, watermark, blurred, soft focus, isometric, low-poly

## Generation tips

- **Midjourney**: append `--ar 1:1 --stylize 250 --v 6.1` to each prompt
- **DALL-E 3**: paste the prompt directly; DALL-E often interprets "cream" as washed-out — be ready to regenerate with the colour explicitly named again in conversation
- **Imagen 3 / Recraft**: best at flat illustration style; usually nails the brief on first try
- **Ideogram**: good for anything that includes "no text" — Ideogram is best at suppressing text artifacts

If the first generation doesn't land:
1. Re-roll 2–3 times — variance is high
2. If the colour palette drifts, re-paste only the palette block in your follow-up
3. If the style drifts toward photorealism, re-emphasise "flat illustration, no gradients" in the negative prompt
4. If the subject is wrong shape, name 1–2 specific reference illustrators in the prompt

When you have a winning generation, save it as `.png` to the path listed at the top of each lesson section. The lesson markdown already references that path — image will render automatically on next page load.
